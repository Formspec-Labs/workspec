**Seam framing.** Runtime ¬ß12 defines nine host-interface seams composed by `wos-runtime`. Endpoint gaps ride on the same seams; wiring seams first unblocks their downstream handlers. See [`PARITY.md`](PARITY.md) ‚ñé Compounding costs of deferral for the full rationale.
